## 6. Технические рекомендации для разработчиков

### 6.1. Базовая SEO-конфигурация

#### robots.txt
```
User-agent: *
Allow: /
Disallow: /admin/
Disallow: /wp-admin/          (если WordPress)
Disallow: /cgi-bin/
Disallow: /?
Disallow: /wp-
Disallow: */feed/
Disallow: */trackback/
Disallow: */comments/
Disallow: */comment-
Disallow: */page/*
Disallow: /*?s=
Disallow: *&

# Яндекc-XML
Sitemap: https://minites.ru/sitemap_index.xml

# Google
Sitemap: https://minites.ru/sitemap.xml
```

#### sitemap.xml
- Автоматическая генерация (WordPress: Yoast SEO / Rank Math)
- Приоритеты: Главная (1.0) → Услуги (0.8) → Оборудование (0.7) → Блог (0.6) → Регионы (0.5)
- Частота обновлений: Ежедневно (блог), Еженедельно (услуги), Ежемесячно (статичные)

### 6.2. Микроразметка Schema.org

| Тип разметки | Где применяется | Поля |
|-------------|-----------------|------|
| **Organization** | Главная, О компании | name, url, logo, address, contactPoint, sameAs |
| **Product** | Страницы оборудования | name, description, brand, offers (price), aggregateRating |
| **Service** | Страницы услуг | name, description, provider, areaServed, hasOfferCatalog |
| **FAQPage** | FAQ-блоки на страницах | mainEntity (Question + acceptedAnswer) |
| **BreadcrumbList** | Все внутренние страницы | itemListElement (position, name, item) |
| **Article** | Страницы блога | headline, datePublished, author, publisher, image |
| **LocalBusiness** | Региональные страницы | name, address, areaServed, geo |
| **WebSite** | Главная (для Sitelinks Search Box) | url, potentialAction (SearchAction) |

### 6.3. Core Web Vitals и скорость

| Метрика | Цель | Способ достижения |
|---------|------|-------------------|
| **LCP** < 2.5 сек | Оптимизация изображений (WebP, lazy load), CDN, кэширование |
| **FID/INP** < 100 мс | Минимализация JS, defer/async для некритичных скриптов |
| **CLS** < 0.1 | Фиксированные размеры изображений, reserve space для шрифтов и ads |
| **TTFB** < 600 мс | Быстрый сервер, кэширование страниц, Varnish/Redis |

### 6.4. Мобильная адаптация

- Responsive design (mobile-first)
- Touch-friendly элементы (минимум 48×48 px)
- Accelerated Mobile Pages (AMP) для статей блога (опционально)
- PWA-манифест для возможности «на экран домой»

### 6.5. Настройка аналитики и целей

#### Яндекс.Метрика (цели)

| Название цели | Условие |
|---------------|---------|
| **CTA_Расчёт** | Клик по кнопке «Рассчитать окупаемость» |
| **CTA_Консультация** | Клик по кнопке «Получить консультацию» |
| **Форма_Отправлена** | Отправка любой формы (success page / event) |
| **Калькулятор_Использован** | Событие использования калькулятора |
| **Скролл_75%** | Просмотр 75% страницы (quality check) |
| **Время_на_сайте_3мин** | Более 180 секунд на сайте |

#### Google Analytics 4 (conversion events)

| Event Name | Параметры |
|------------|-----------|
| `generate_lead` | form_name, page_location |
| `calculator_use` | power_kw, region, result_savings |
| `scroll_depth` | percent_scrolled, page_title |
| `file_download` | file_name, file_type (PDF-кейсы, Excel-расчёты) |
| `phone_click` | phone_number (call tracking) |

### 6.6. Безопасность и индексация

- **HTTPS** — обязательно (Let's Encrypt)
- **Canonical URLs** — на всех страницах (избежать дублей с/без www, http/https, trailing slash)
- **Hreflang** — если планируется версия на английском (для СНГ-рынков)
- **404-страница** — кастомная с поиском и популярными услугами
- **X-Robots-Tag** — noindex для служебных страниц (спасибо, поиск, корзина если будет)
