### Выявленные ценовые модели:

#### 1. МОДЕЛЬ "ВЫКУП ПО ОБЪЕМУ ЭЭ"
**Суть:** Клиент платит за фактически потребленную электроэнергию (кВт*ч)
- Тариф формируется из нескольких компонентов
- После выкупа оборудования тариф снижается
- Пример: "заказчик будет платить по тарифу от ГПУ... после выкупа ГПУ"

**Структура тарифа:**
- Расходы на газ: `0,3м3 × мощность × цена газа × 24ч × 30дней`
- Обслуживание ГПУ: `1,4р × кВт*ч`
- Выплата инвестору: фиксированная сумма или % от разницы тарифов
- Выплата подрядчику: фиксированная сумма

#### 2. МОДЕЛЬ "ЛИЗИНГ С ФИКСИРОВАННЫМ ТАРИФОМ"
**Суть:** Клиент платит фиксированный тариф в течение срока лизинга
- Аванс: ~5-10% от стоимости оборудования
- Ежемесячные платежи по лизингу
- После лизинга оборудование переходит в собственность
- Пример: "фиксированный тариф по лизингу 8.91р/кВт*ч"

**Структура расходов:**
- Аванс
- Оплата по лизингу
- Оплата за газ
- Обслуживание
- Экономия на налогах

#### 3. МОДЕЛЬ "ЭНЕРГОСЕРВИСНЫЙ КОНТРАКТ"
**Суть:** Комплексное обслуживание с оплатой за результат
- Инвестор владеет оборудованием
- Клиент платит только за электроэнергию
- Инвестор получает доход из разницы между тарифом ГПУ и сетевым тарифом
- Пример: "выплата инвестору за 2й блок (из разницы по тарифу)"

### Ключевые финансовые показатели из расчетов:

| Параметр | Значение | Примечание |
|----------|----------|------------|
| Обслуживание ГПУ | 1,4 руб/кВт*ч | Фиксированная ставка |
| Расход газа | 0,3 м3/кВт*ч | Зависит от нагрузки |
| Тариф на газ | 7,35-10,88 руб/м3 | Разный для разных регионов |
| Сетевой тариф на ЭЭ | 4,0-11,46 руб/кВт*ч | Сильно варьируется |
| КИУМ (коэфф. использования) | 0,8 | Для майнинг-ферм |
| Мощность установок | 90-3000 кВт | От малых до промышленных |

### Структура цены электроэнергии для клиента:

```
ТАРИФ НА ЭЭ ОТ ГПУ = 
  + Расходы на газ (0,3 м3 × цена газа)
  + Обслуживание ГПУ (1,4 руб/кВт*ч)
  + Выплата инвестору (фиксированная или %)
  + Выплата подрядчику (фиксированная)
  + Накладные расходы
  + Прибыль
```

## ШАГ 4: ВЫВОДЫ И РЕКОМЕНДАЦИИ

### 3 КЛЮЧЕВЫЕ ФУНКЦИИ САЙТА (финансовая направленность):

#### 1. КАЛЬКУЛЯТОР СТОИМОСТИ ЭЛЕКТРОЭНЕРГИИ
**Назначение:** Позволить клиенту рассчитать свою выгоду
**Функционал:**
- Ввод: мощность, часы работы, текущий тариф от сети
- Вывод: стоимость от ГПУ с разбивкой по компонентам
- Сравнение: с сетевым тарифом и покупкой оборудования
- Детализация: газ, обслуживание, инвестор, подрядчик

**Пример структуры:**
```
Ваш тариф от ГПУ: X руб/кВт*ч
├── Расходы on gas: Y руб/кВт*ч (0,3м3 × цена газа)
├── Обслуживание: 1,4 руб/кВт*ч
├── Выплата инвестору: Z руб/кВт*ч
├── Выплата подрядчику: W руб/кВт*ч
└── Итого: X руб/кВт*ч

Экономия vs сеть: (сетевой тариф - X) × объем = экономия в месяц
```

#### 2. СТРУКТУРА ЦЕНЫ (образовательный раздел)
**Назначение:** Объяснить, из чего складывается цена электроэнергии
**Содержание:**
- **Аренда оборудования (ГПУ)** - амортизация, лизинг
- **Обслуживание** - ТО, ремонты, запчасти
- **Расход газа** - объем × тариф
- **Выплаты инвестору** - доход на вложенный капитал
- **Выплаты подрядчику** - монтаж, ПНР, сервис
- **Накладные расходы** - логистика, управление
- **Прибыль** - маржинальность проекта

**Формат:** Инфографика + подробные пояснения по каждому пункту

#### 3. СРАВНЕНИЕ С АЛЬТЕРНАТИВАМИ
**Назначение:** Показать преимущества аренды перед другими вариантами
**Сравнение:**
- **Аренда vs Сеть** - экономия, независимость, надежность
- **Аренда vs Покупка** - нет CAPEX, сервис включен, гибкость
- **Аренда vs Дизель-генератор** - дешевле, экологичнее, тише

**Формат:** Таблицы, графики, реальные примеры расчетов

### РЕКОМЕНДУЕМАЯ ЦЕНОВАЯ МОДЕЛЬ ДЛЯ САЙТА:

**Базовая модель: "АРЕНДА С ВЫКУПОМ ПО ОБЪЕМУ ЭЭ"**

**Преимущества:**
1. **Низкий порог входа** - нет крупных первоначальных вложений
2. **Прозрачность** - клиент платит только за потребленную энергию
3. **Гибкость** - можно масштабировать под changing needs
4. **Предсказуемость** - фиксированный тариф на срок контракта
5. **Выгоднее сети** - экономия 20-40% относительно сетевых тарифов

**Структура тарифа для отображения на сайте:**
```
ТАРИФ НА ЭЛЕКТРОЭНЕРГИЮ ОТ ГПУ

1. Расходы на газ: 0,3 м3 × [текущая цена газа в регионе]
2. Обслуживание ГПУ: 1,4 руб/кВт*ч (фиксировано)
3. Выплата инвестору: [рассчитывается индивидуально]
4. Выплата подрядчику: [рассчитывается индивидуально]

ИТОГО: от X руб/кВт*ч (зависит от мощности и региона)

Срок контракта: 3-5 лет
После выкупа: тариф снижается на 30-40%
```

**Дополнительные опции:**
- Лизинг с последующим выкупом
- Энергосервисный контракт (оплата за результат)
- Гибридные решения (ГПУ + сеть + аккумуляторы)
