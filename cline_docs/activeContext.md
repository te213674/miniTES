### Завершенные работы (Этап 3 - Разработка калькулятора):

1. **Калькулятор энергосервисного контракта** (`calculator/public/index.html`):

   - Расчёт тарифа с учётом майнинг-ферм

   - Мета-теги (Title, Description, Keywords)

   - Open Graph теги для соцсетей

4. **Формы захвата лидов**:
   - Модальное окно "Получить детальный расчёт"
   - Сохранение заявок в БД (таблица `leads`)
   - API endpoint `/api/calculator/lead`

5. **Админ-панель** (`calculator/public/admin.html`):
   - Аутентификация (JWT токены)
   - Страница входа `/admin/login`
   - Управление оборудованием (CRUD)
   - Управление майнинг-фермами
   - Просмотр и управление лидами

6. **Настройка CORS**:
   - Разрешены запросы с домена основного сайта
   - Настроены заголовки CORS в Express
