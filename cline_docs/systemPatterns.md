# System Patterns: Архитектура и технические решения

## Общее описание системы

**miniTES.ru** - это веб-платформа для продвижения услуг в области распределенной энергетики, специализирующаяся на газопоршневых электростанциях (ГПУ) и мини-ТЭЦ.

## Ключевые технические решения

### 1. Архитектура сайта

**Текущее состояние:**
- Сайт требует модернизации и оптимизации
- Необходимо улучшение онлайн-присутствия (по результатам конкурентного анализа)

**Целевая архитектура:**

```
Frontend (HTML/CSS/JS)
    ↓
CMS/Платформа (рекомендуется WordPress или кастомное решение)
    ↓
Backend (PHP/Python/Node.js)
    ↓
База данных (MySQL/PostgreSQL)
    ↓
Внешние API (аналитика, формы, калькуляторы)
```

### 2. Основные компоненты

#### A. Информационные страницы
- **Главная** - УТП, преимущества, призывы к действию
- **Услуги** - проектирование, поставка, монтаж, сервис
- **Оборудование** - каталог ГПУ и мини-ТЭЦ
- **Кейсы** - примеры реализованных проектов
- **Блог** - технические статьи, аналитика, новости
- **Калькулятор** - расчет окупаемости мини-ТЭЦ
- **Контакты** - форма связи, реквизиты

#### B. Функциональные модули
- **Калькулятор окупаемости** - интерактивный расчет ROI
- **Формы захвата** - заявки на расчет, консультацию
- **CRM-интеграция** - управление лидами
- **Аналитика** - Яндекс.Метрика, Google Analytics
- **SEO-модуль** - оптимизация под поисковые системы

### 3. Технологический стек (рекомендации)

**На основе анализа конкурентов:**

| Компонент | 1vat.ru (конкурент) | Рекомендация для miniTES.ru |
|-----------|---------------------|----------------------------|
| CMS | Кастомная разработка | WordPress (гибкость, SEO) |
| Аналитика | GA + Яндекс.Метрика | GA + Яндекс.Метрика + цели |
| Мобильная версия | Адаптирована | Адаптивная + PWA |
| SSL | Есть | Обязательно |
| SEO | robots.txt + sitemap.xml | + микроразметка schema.org |

### 4. Паттерны проектирования

#### A. User Experience (UX)
- **Простая навигация** - не более 3 кликов до целевого действия
- **Адаптивный дизайн** - мобильная версия обязательна
- **Быстрая загрузка** - оптимизация изображений, кэширование
- **Четкие CTA** - "Рассчитать окупаемость", "Получить консультацию"

#### B. Content Strategy
- **Образовательный контент** - статьи о преимуществах когенерации
- **Кейсы с цифрами** - реальная экономия клиентов
- **Технические расчеты** - прозрачность и экспертность
- **FAQ** - ответы на частые вопросы

#### C. Conversion Optimization
- **Множественные точки захвата** - формы в шапке, в контенте, popup
- **Социальные доказательства** - отзывы, логотипы клиентов
- **Гарантии** - сроки окупаемости, качество оборудования
- **Онлайн-консультант** - в отличие от 1vat.ru (у них нет)

### 5. Интеграции и API

**Необходимые интеграции:**
- **Яндекс.Метрика** - аналитика посещаемости
- **Google Analytics** - дополнительная аналитика
- **CRM система** - управление заявками (AmoCRM, Bitrix24)
- **Email рассылка** - уведомления о новых заявках
- **Телефония** - отслеживание звонков (Call-tracking)
- **Поисковые системы** - Яндекс.Вебмастер, Google Search Console

### 6. Безопасность и производительность

**Требования:**
- **SSL сертификат** - обязательно (HTTPS)
- **Резервное копирование** - ежедневное
- **Защита от DDoS** - базовая защита
- **Оптимизация скорости** - загрузка < 3 секунд
- **Мобильная оптимизация** - Google Mobile-Friendly

### 7. Масштабируемость

**План развития:**
1. **Этап 1** - Базовый сайт с калькулятором и формами
2. **Этап 2** - Личный кабинет клиента (мониторинг объектов)
3. **Этап 3** - Мобильное приложение для клиентов
4. **Этап 4** - Интеграция с IoT (удаленный мониторинг ГПУ)

## Стандарты кода и разработки

- **Чистый код** - соблюдение стандартов (PSR для PHP, ESLint for JS)
- **Версионирование** - Git, семантическое версионирование
- **Тестирование** - unit-тесты, интеграционное тестирование
- **Документация** - комментарии, README, техническая документация
- **Code Review** - обязательная проверка кода

## Мониторинг и аналитика

**Ключевые метрики:**
- **Посещаемость** - уникальные посетители, глубина просмотра
- **Конверсия** - заявки, звонки, расчеты в калькуляторе
- **SEO** - позиции по ключевым запросам, органический трафик
- **Скорость** - время загрузки страниц, Core Web Vitals
- **Мобильность** - доля мобильных пользователей, удобство
